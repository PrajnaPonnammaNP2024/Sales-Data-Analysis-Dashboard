# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data to identify key business insights using Power BI. The dashboard provides a clear view of revenue, profit, product performance, and monthly sales trends.

## Tools Used
- Power BI
- Excel
- SQL

## Key KPIs
- Total Revenue
- Total Profit
- Top Selling Products
- Monthly Sales Trend
- Sales by Region

## Dashboard Features
- Interactive visualizations
- KPI cards for quick business insights
- Sales trend analysis
- Product performance comparison

## Key Insights
- Certain product categories contribute the highest revenue.
- Sales trends show seasonal patterns across months.
- Some regions generate higher profit compared to others.

## Dataset
The dataset used for this project was obtained from Kaggle.
